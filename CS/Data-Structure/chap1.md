## 算法和算法评价

### 算法的基本概念

对特定问题求解步骤的描述

五个重要特征：有穷性、确定性、可行性、输入、输出

“好”算法目标：正确性、可读性、健壮性、高效率与低存储量需求

### 算法效率的度量

#### 时间复杂度

主要分析$T(n)$的数量级

语句的频度：语句的重复执行次数

$T(n)$：所有语句频度之和

基本运算（最深层循环内的语句）的频度与$T(n)$同数量级

算法的时间复杂度记为
$$
T(n) = O(f(n)) \\
C \gt 0, 当 n \ge n_0 时, 都满足 0 \le T(n) \le Cf(n)
$$
