## 重点回顾

### 堆排序

#### 建堆

按照完全二叉树顺序插入结点，比大小进行交换

（2019-11）如给定序列，需要调整，则最下一层开始，从左到右

#### 调整

根结点移除之后，将最右下位置放到根的位置，然后比较大小进行调整
